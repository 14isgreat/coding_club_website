GitHub

A way to save your files and work with others

Difficulty level: 5/10

What is GitHub?

GitHub is an online platform where developers store, share, and collaborate on code.

It uses version control, allowing users to track changes, manage projects, and work together efficiently.

GitHub makes it easy to contribute to open-source software, review code, and keep a history of all updates and improvements.

Website link: github.com

How to push (save) your project to GitHub

1. Go to terminal ( Terminal > New Terminal if no terminal is open )

2. Initialize git

git init

3. Replace username and repository with your actual GitHub username and repository name:

git remote add origin https://github.com/username/repository.git

4. The . stages all files in the directory. You can also add specific files.

git add .

5. Commit Your Changes

git commit -m "Your commit message"

6. Push to GitHub

git push -u origin main

Now you can go to your repository in GitHub, the changes you made should be visible there

How to pull code from a GitHub repository

1. Go to terminal ( Terminal > New Terminal if no terminal is open )

2. Initialize git

git init

3. If you're pulling the code for the first time:

git clone https://github.com/username/repo-name.git

4. If you already cloned the repo earlier, run this inside the repo folder to get the newest version:

git pull

That's it!
