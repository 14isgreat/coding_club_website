What is Python?

Tutorial: Hello, World!

The classic first python program is getting "Hello, World!" displayed in the terminal.

To do so, copy-paste the following line in your program:

print("Hello, World!")

Notice that the message you are displaying is enveloped in brackets and quotation marks. (" your message ")

If you don't do this, you will get an error message!

Click the run button, located in the top right of your screen.

You should now see Hello, World! in the terminal. Hooray! You just made your first python program.

Tutorial: Variables and Data Types

1. Variables are a way of saving data. You can assign different values to variables, including numbers and text.

Here is an example. Copy it into your code:

x = 5 y = "Hello, Python!" z = 3.14

In Python terminology, you call text "string", numbers "integers", and decimals "floats".

Notice that strings have " " around them (that's why we needed the quatation marks while printing "Hello, World!"), while numbers don't.

2. Now that you have these variables, you can use them.

What do you think will be displayed in the terminal if you run the following line?

print(x+z)

It should display 8.14. Why? Because if x = 5 and z = 3.14 , x + y = 5 + 3.14 = 8.14!

Tutorial: Conditional Statements

1. If-else statements are a way of checking if something is true.

x = 10 if x > 5: print("x is greater than 5") else: print("x is less than or equal to 5")

In this example, since x > 5, it will do whatever is in the next line. In this case print x is greater than 5.

Make sure that the next line afterwards is indented, otherwise you will get an error message!

Tutorial: Loops

With loops you can get something to repeat a certain amount of times. There are different types of loops.

1. For loop:

for i in range(5): print(i)

This loop prints the variable i 5 times.

In For loops, you must type "for", then create a variable, in this case "i" (short for iteration) and then "in range():".

In between the brackets () you must put the amount of times you want it to repeat.

2. While loop:

x = 0 while x < 5: print(x) x = x + 1 # because of this line, 1 is added to x each time it repeats

With While loops you must type "while" and afterwards there must be a condition. In this case: x < 5

While loops repeat forever, until the condition is no longer met.

Be careful, if your condition is never met your program will run forever!

If you really want something to run forever:

while True: # True will always be True, so this runs forever print("hello")

Tutorial: Functions

Functions are a way of saving code.

For example, if you want something to happen every now and then, but not in a way where you can just put it in a loop.

1. Define and call a function:

def greet(name): return f"Hello, {name}!" greeting = greet("Alice") print(greeting)

Tutorial: Lists and Dictionaries

With lists and dictionaries you can also save information in a structured manner.

1. Lists:

fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit)

2. Dictionaries:

person = {"name": "Alice", "age": 25} print(person["name"]) print(person["age"])

Tutorial: Classes and Objects

1. Define a class and create an object:

class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): return f"Hi, I'm {self.name} and I'm {self.age} years old." person = Person("Alice", 25) print(person.greet())

Tutorial: File Handling

1. Read a file:

with open("example.txt", "r") as file: print(file.read())

2. Write to a file:

with open("example.txt", "w") as file: file.write("Hello, Python!")

Tutorial: Error Handling

1. Use try-except:

try: result = 10 / 0 except ZeroDivisionError: print("You can't divide by zero!")
