CSS

Cascading Style Sheets for Web Design

Difficulty level: 4/10

What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance and layout of web pages. It allows you to style HTML elements, including colors, fonts, spacing, and more.

Setting Up

1. Create a file named styles.css in your project folder.

2. Link it to your HTML file by adding the following line in the <head> section:

<link rel="stylesheet" href="styles.css">

Tutorial: Selectors

1. Styling by tag name:

h1 { color: blue; font-size: 36px; }

2. Styling by class:

.highlight { background-color: yellow; padding: 5px; }

3. Styling by ID:

#main-title { text-align: center; font-weight: bold; }

Tutorial: Colors

1. Use predefined color names:

body { background-color: lightgray; }

2. Use hexadecimal, RGB, or HSL values:

p { color: #ff5733; background-color: rgb(240, 240, 240); }

Tutorial: Fonts

1. Change the font family:

body { font-family: Arial, sans-serif; }

2. Adjust font size and weight:

h1 { font-size: 48px; font-weight: 700; }

Tutorial: Box Model

1. Add padding and margin:

div { padding: 10px; margin: 15px; }

2. Add a border:

div { border: 2px solid black; border-radius: 10px; }

3. Set width and height:

div { width: 200px; height: 100px; }

Tutorial: Flexbox

1. Basic Flexbox container:

.flex-container { display: flex; justify-content: center; align-items: center; gap: 20px; }

2. Individual flex item properties:

.flex-item { flex: 1; }

Tutorial: Grid

1. Define a grid container:

.grid-container { display: grid; grid-template-columns: 1fr 1fr; gap: 10px; }

2. Position grid items:

.grid-item { grid-column: span 2; }

Tutorial: Transitions

1. Smooth hover effects:

button { transition: background-color 0.3s ease; } button:hover { background-color: darkblue; }

2. Animations:

@keyframes fadeIn { from { opacity: 0; } to { opacity: 1; } } div { animation: fadeIn 2s; }
