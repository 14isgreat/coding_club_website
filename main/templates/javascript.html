JavaScript

The Language of the Web

Difficulty level: 6/10

What is JavaScript?

JavaScript is a versatile programming language used to create interactive websites, server-side applications, and more.

It is an essential technology for modern web development, alongside HTML and CSS.

Setting Up

1. In the HTML file you are working on add the following code in the head:

<script src="script.js"></script>

2. Create a file named script.js and add your JavaScript code there.

Tutorial: Hello, World!

1. Output "Hello, World!" to the console:

console.log("Hello, World!");

2. Output it to the browser window:

document.write("Hello, World!");

Tutorial: Variables and Data Types

1. Declare variables:

let x = 5; const y = "Hello, JavaScript!"; var z = 3.14;

2. Log the data type of variables:

console.log(typeof x); console.log(typeof y); console.log(typeof z);

Tutorial: Conditional Statements

1. If-else example:

let number = 10; if (number > 5) { console.log("Number is greater than 5"); } else { console.log("Number is 5 or less"); }

Tutorial: Loops

1. For loop:

for (let i = 0; i < 5; i++) { console.log(i); }

2. While loop:

let i = 0; while (i < 5) { console.log(i); i++; }

Tutorial: Functions

1. Define and call a function:

function greet(name) { return `Hello, ${name}!`; } console.log(greet("Alice"));

Tutorial: Arrays and Objects

1. Arrays:

let fruits = ["apple", "banana", "cherry"]; fruits.forEach(fruit => console.log(fruit));

2. Objects:

let person = { name: "Alice", age: 25 }; console.log(person.name); console.log(person.age);

Tutorial: DOM Manipulation

1. Modify the content of an HTML element:

document.getElementById("myElement").innerText = "New Content";

2. Add a click event listener:

document.getElementById("myButton").addEventListener("click", function() { alert("Button clicked!"); });

Tutorial: Classes

1. Define a class and create an object:

class Person { constructor(name, age) { this.name = name; this.age = age; } greet() { return `Hi, I'm ${this.name} and I'm ${this.age} years old.`; } } let person = new Person("Alice", 25); console.log(person.greet());
