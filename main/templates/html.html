HTML

Markup Language for Web Development

Difficulty level: 3/10

What is HTML?

HTML (HyperText Markup Language) is the standard language for creating web pages and applications. It structures the content of a webpage using a series of elements (tags).

These tags tell the browser how to display the text, images, links, and other multimedia content.

Setting Up

1. Create a file named index.html in your project folder.

2. Write the basic structure of an HTML file:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Page</title> </head> <body> <h1>Hello, world!</h1> </body> </html>

Tutorial: Adding Text

1. Headings (from largest to smallest):

<h1>This is a Heading 1</h1> <h2>This is a Heading 2</h2> <h3>This is a Heading 3</h3>

2. Paragraphs:

<p>This is a paragraph of text.</p>

3. Inline formatting:

<strong>Bold text</strong> and <em>italic text</em>.

Tutorial: Adding Links

1. Basic link:

<a href="https://www.example.com">Visit Example</a>

2. Open link in a new tab:

<a href="https://www.example.com" target="_blank">Open in New Tab</a>

Tutorial: Adding Images

1. Adding an image:

<img src="path/to/image.jpg" alt="Description of the image">

2. Adding a caption with figure:

<figure> <img src="path/to/image.jpg" alt="Description"> <figcaption>This is the image caption.</figcaption> </figure>

Tutorial: Forms

1. Basic form structure:

<form action="/submit" method="POST"> <label for="name">Name:</label> <input type="text" id="name" name="name"> <button type="submit">Submit</button> </form>

2. Add a dropdown:

<select name="options"> <option value="1">Option 1</option> <option value="2">Option 2</option> </select>

Tutorial: Using Divs for Layout

1. Create a container:

<div class="container"> <h2>Title</h2> <p>This is a container for content.</p> </div>

2. Apply CSS for layout:

.container { margin: 0 auto; max-width: 800px; padding: 20px; background-color: #f9f9f9; }

Tutorial: Embedding Media

1. Embedding a video:

<video controls> <source src="video.mp4" type="video/mp4"> Your browser does not support the video tag. </video>

2. Embedding an audio file:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Your browser does not support the audio tag. </audio>

Tutorial: Advanced Forms

1. File upload:

<form action="/upload" method="POST" enctype="multipart/form-data"> <input type="file" name="file"> <button type="submit">Upload</button> </form>

2. Validation with required:

<input type="text" name="username" required>
