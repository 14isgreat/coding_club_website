Java

The Language for Enterprise Applications

Difficulty level: 7/10

What is Java?

Java is a versatile, platform-independent programming language.

It runs on the Java Virtual Machine (JVM), enabling "Write Once, Run Anywhere" (WORA).

Its strengths include security, multithreading, garbage collection, and its extensive standard library.

Setting Up

1. Download and install the Java Development Kit (JDK) from the official website.

2. Set up your IDE. Common choices include IntelliJ IDEA, Eclipse, or VS Code with a Java extension.

3. Verify installation by typing java -version and javac -version in the terminal.

4. Create a file called HelloWorld.java and add the following code:

public class HelloWorld { public static void main(String[] args) { System.out.println("Hello, World!"); } }

5. Compile and run the file using:

javac HelloWorld.java java HelloWorld

Tutorial: Variables and Data Types

1. Declare variables:

int number = 10; String text = "Hello, Java!"; double pi = 3.14;

2. Print their values:

System.out.println(number); System.out.println(text); System.out.println(pi);

Tutorial: Conditional Statements

1. If-else example:

int age = 20; if (age >= 18) { System.out.println("You are an adult."); } else { System.out.println("You are not an adult."); }

Tutorial: Loops

1. For loop:

for (int i = 0; i < 5; i++) { System.out.println("Count: " + i); }

2. While loop:

int i = 0; while (i < 5) { System.out.println("Count: " + i); i++; }

Tutorial: Functions

1. Define and call a method:

public class Main { public static void main(String[] args) { greet("Alice"); } public static void greet(String name) { System.out.println("Hello, " + name + "!"); } }

Tutorial: Arrays

1. Declare and print an array:

int[] numbers = {1, 2, 3, 4, 5}; for (int number : numbers) { System.out.println(number); }

Tutorial: Classes and Objects

1. Define a class and create an object:

public class Person { String name; int age; public Person(String name, int age) { this.name = name; this.age = age; } public void introduce() { System.out.println("Hi, I'm " + name + " and I'm " + age + " years old."); } } public class Main { public static void main(String[] args) { Person person = new Person("Alice", 25); person.introduce(); } }

Tutorial: Inheritance

1. Create a subclass:

class Animal { void eat() { System.out.println("This animal eats food."); } } class Dog extends Animal { void bark() { System.out.println("This dog barks."); } } public class Main { public static void main(String[] args) { Dog dog = new Dog(); dog.eat(); dog.bark(); } }

Understanding the JVM

The JVM (Java Virtual Machine) is an essential component of Java's platform independence. When you compile Java code, it generates .class files containing bytecode. The JVM interprets and executes this bytecode on any platform.

1. Compile your code:

javac Main.java

2. Execute the bytecode with the JVM:

java Main

Memory Management: Garbage Collection

Java has an automatic garbage collector that reclaims memory occupied by objects no longer in use. You can request garbage collection manually (though it's not guaranteed to run immediately):

System.gc();

Exception Handling

Java uses a robust system of exceptions to handle errors. Use try, catch, and finally blocks to manage exceptions:

try { int result = 10 / 0; // Will throw ArithmeticException } catch (ArithmeticException e) { System.out.println("Cannot divide by zero: " + e.getMessage()); } finally { System.out.println("Execution completed."); }

Multithreading

Java's Thread class and Runnable interface make it simple to write multithreaded programs. Here's an example of creating and running threads:

class MyThread extends Thread { public void run() { System.out.println("Thread is running."); } } public class Main { public static void main(String[] args) { MyThread thread = new MyThread(); thread.start(); } }

Or, using Runnable:

class MyRunnable implements Runnable { public void run() { System.out.println("Runnable is running."); } } public class Main { public static void main(String[] args) { Thread thread = new Thread(new MyRunnable()); thread.start(); } }

Interfaces

Unlike classes, interfaces can only define abstract methods and constants (before Java 8). A class can implement multiple interfaces:

interface Animal { void sound(); } class Dog implements Animal { public void sound() { System.out.println("Woof!"); } } public class Main { public static void main(String[] args) { Animal dog = new Dog(); dog.sound(); } }

Generics

Generics provide type safety for collections and other classes. Here's an example with a generic method:

import java.util.ArrayList; public class Main { public static void main(String[] args) { ArrayList list = new ArrayList<>(); list.add("Java"); list.add("Programming"); for (String item : list) { System.out.println(item); } } }

Annotations

Annotations provide metadata about your code. Common annotations include @Override and @Deprecated:

class Parent { void display() { System.out.println("Parent class"); } } class Child extends Parent { @Override void display() { System.out.println("Child class"); } }

Project: Banking System

Create a simple banking system with the following requirements:

  • A BankAccount class with methods to deposit, withdraw, and check balance.
  • Exception handling for invalid operations (e.g., overdraft).
  • Multiple accounts managed in an ArrayList.

Sample code:

class BankAccount { private double balance; public BankAccount(double initialBalance) { this.balance = initialBalance; } public void deposit(double amount) { balance += amount; } public void withdraw(double amount) throws Exception { if (amount > balance) { throw new Exception("Insufficient funds."); } balance -= amount; } public double getBalance() { return balance; } } public class Main { public static void main(String[] args) { BankAccount account = new BankAccount(1000); try { account.withdraw(1200); } catch (Exception e) { System.out.println(e.getMessage()); } System.out.println("Balance: " + account.getBalance()); } }
