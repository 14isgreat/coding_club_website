What is Django?

This website was created with Django!

Django is like a toolkit that helps you build websites quickly and efficiently. Imagine you want to create a website, like an online store or a blog. Instead of starting from scratch and figuring out how to make everything work (like user accounts, managing data, or handling payments), Django gives you pre-built tools to make it easier.

Think of it as a set of LEGO pieces for websites. Each piece has a specific job, like organizing information, showing it to visitors, or letting people log in securely. You just put the pieces together to create the website you want.

It’s built with Python, a user-friendly programming language, and is known for being reliable and easy to use. Django also focuses on making websites secure, fast, and scalable—so they can handle lots of visitors.

One of Django’s big strengths is its ability to manage data. For example, if you’re creating a blog, Django makes it easy to store, update, and display your articles.

In short, Django is a powerful, beginner-friendly framework that handles the complex stuff behind the scenes, so you can focus on making your website look and work the way you want. It’s perfect for turning ideas into reality!

See the Official Documentation or continue reading to learn this framework!

Starting with a template

1. Open Visual Studio Code

2. Open the coding-club folder

3. Run this command in the terminal ( Terminal > New Terminal if no terminal is open ):

git clone https://github.com/14isgreat/django_template.git

4. Rename the project (No spaces can be in the name, you can use _ instead) by runnning:

mv django_template {replace with your website name}

5. The website has been created. Navigate to the project file by running:

cd {replace with whatever you renamed your project to}

6. If Django is not installed yet, run this command: (You can run it just in case)

pip install django

7. You are all set up! To open the website run:

python manage.py runserver

... and then open the link which will pop up in the terminal in a few seconds (Looks something like: http://127.0.0.1:8000/ )

8. You can now click here to skip down to the tutorial

Starting from scratch

1. Open Visual Studio Code

2. Open the coding-club folder

3. Run the following commands in the terminal ( Terminal > New Terminal if no terminal is open ):

pip install django
django-admin startproject {replace with your projectname}
cd {replace with your projectname}

4. Start the development server to make sure everything is working:

python manage.py runserver

Visit http://127.0.0.1:8000/ in your web browser. You should see the Django "It worked!" page.

5. Now run the following command in the terminal:

python manage.py startapp main

6. Edit the INSTALLED_APPS list in settings.py by adding 'main', to the end. It should look like this:

INSTALLED_APPS = [ # other apps 'main', ]

7. In the main folder create a file called urls.py. In it, write:

from django.urls import path from . import views # Import views from the current app urlpatterns = [ path('', views.home, name='home'), # Home page view path('about/', views.about, name='about'), ]

8. There is also an existing urls.py file in your other folder. Replace its contents with:

from django.contrib import admin from django.urls import path, include # include is used to include app URLs urlpatterns = [ path('admin/', admin.site.urls), # Admin panel URL path('', include('main.urls')), # Include the app's URLs ]

9. In the main folder, edit the views.py file so that it says:

from django.shortcuts import render def home(request): return render(request, 'home.html') # Render the home.html template def about(request): return render(request, 'about.html') # Render the about.html template

10. Now create two folders that are located inside of the main folder. Name them templates and static

11. Create two files in your templates folder. Name them home.html and about.html. These are your html files. The structure of the files should look similar to:

{% verbatim %} <!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Home</title> {% load static %} <link rel="stylesheet" href="{% static 'home.css' %}"> </head> <body> <h1>Welcome to the Home Page!</h1> <p>This is your home page with its own CSS file.</p> <a href="/about/">Go to About Page</a> </body> </html> {% endverbatim %}

12. Create two files in your css folder. Name them home.css and about.css. These are your css files.

Tutorial: Summary

Firstly you must understand Django's general structure. You have your html files in the templates folder while your css files are in the static folder. Any images you save and want to display in your website and any javascript files you have are also saved in the static folder.

Keep in mind that there are some things that are Django specific that you need to make sure you have. For example, when connecting your html and css files, you must load the static tag. See this for an example.

In the urls.py file inside of the main folder is where you define all the links of your website's pages. When you create a new page you must also update the views.py file. See this for an example.

If you want to make a simple website using html, css, and javascript, you are good to go!

Advanced Tutorial: User Interactions

You can take your website to the next level by enabling interactions with the user. For this, you need to use python.

To do this, you must first create a file called forms.py inside of the main folder.

To ask users a simple question on your website which they can answer, copy this into forms.py:

{% verbatim %} class TextboxForm(forms.Form): color = forms.CharField( label="Textbox", max_length=100000, widget=forms.Textarea(attrs={'class': 'color'}) ) animal = forms.CharField( label="Textbox", max_length=100000, widget=forms.Textarea(attrs={'class': 'animal'}) ) def clean(self): cleaned_data = super().clean() return cleaned_data {% endverbatim %}

...add the following to the beginning of views.py:

{% verbatim %} from .forms import * {% endverbatim %}

...add the following to your home function or whichever page you want to have the interaction on:

{% verbatim %} def home(request): if request.method == 'POST': form = TextboxForm(request.POST) if form.is_valid(): # you can do something with the user's response here, for example save it into a file or database. return redirect('/') # Redirect to homepage else: print(form.errors) messages.error(request, "Please correct the errors below.") else: form = TextboxForm() return render(request,'home.html', {'form': form}) {% endverbatim %}

...and add the following into the html file you want to update, while you can update your css to make the website look nicer:

{% verbatim %} <div id="container"> <h1 id="header">A Quick Questionnaire</h1> <form method="post" action=""> {% csrf_token %} <p class="user-input">What is your favorite color:</p> {{ form.color }} <p class="user-input">What is your favorite animal:</p> {{ form.animal }} <input type="submit" value="Submit" id="submit-button"> </form> </div> {% endverbatim %}
