Turtle

Pattern Making Python Library

Difficulty level: 2/10

What is Turtle?

Turtle is a beginner-friendly Python library used to create drawings and animations.

It provides a virtual "turtle" that moves on the screen, drawing lines as it goes.

You control its movements (like forward, backward, turn), making it ideal for learning programming concepts while creating fun, visual art or designs.

Setting Up

1. Make a new file in the folder coding-club.

Optional: Make a folder called turtle_codes in the coding-club folder and add your file there.

2. The first line of your file:

import turtle as t

3. The last line of your file:

t.done()

Tutorial: First Movements

1. Basic forward and backward movement

t.forward(100) # 100 pixels t.backward(200) # 200 pixels

This does exactly the same, but it's faster to write.

t.fd(100) t.bk(200)

2. Turn left or right

t.right(90) # 90 degrees t.left(45) # 45 degrees

Again, you can shorten your code

t.rt(90) t.lt(45)

Tutorial: Get rid of Turtle *shocking*

You don't like turtle? Make him go away

t.hideturtle()

Regret your decision? I thought so. Bring him back.

t.showturtle()

You can also change the shape. Bring back the real turtle.

t.shape("turtle")

There are a few different shapes to choose from. They are “arrow”, “turtle”, “circle”, “square”, “triangle”, “classic”. But none of them are as cool as turtle.

You can also make custom shapes. But making those is complicated. If you want, here is how to do it. Not recommended.

Tutorial: First Shapes

1. Drawing a square

t.fd(100) t.rt(90) t.fd(100) t.rt(90) t.fd(100) t.rt(90) t.fd(100) t.rt(90)

You can shorten the code by adding a repetition, since earlier you are simply repeating two lines four times.

for i in range(4): # this makes the code underneath (which needs to be indented) repeat four times t.fd(100) t.rt(90)

2. Drawing a triangle

t.fd(100) t.lt(120) t.fd(100) t.lt(120) t.fd(100) t.lt(120)

A triangle has three angles with a total of 180 degrees -> for each side length to be the same you need an angle of 60 -> therefore the pen has to move (180-60=120) degrees

You can also shorten this code.

for i in range(3): t.fd(100) t.lt(120)

PROJECT: A program which draws three squares next to each other.

ADVANCED: Use a function to create each square and let the user choose how many squares should be drawn next to each other.

EXPERT: Instead of a square every time, a random shape is drawn, ranging from a triangle (3 corners) to a shape with 100 corners.

Tutorial: Colors

1. Changing the pen color

t.pencolor("yellow")

You can also pass RGB values.

t.pencolor(83, 58, 27)

2. Fill in a shape with a color

t.pencolor("red") t.begin_fill() for i in range(4): # this makes the code underneath (which needs to be indented) repeat four times t.fd(100) t.rt(90) t.end_fill()

PROJECT: Each square from your previous project is a different color.

ADVANCED: Pass the color of each square through the function. Make the color of each square be randomly chosen.

EXPERT: Each shape has as many colors as there are corners. No color features more than once.

Tutorial: Picking the pen up

1. To move the turtle without leaving a trail, there is a simple command

t.penup()

2. To place it back down you use this command

t.pendown()

PROJECT: Leave a space between each square, so that it really looks like there are three squares next to each other.

ADVANCED: The turtle starts drawing at the top of the screen and draws squares in rows of 5, before moving down to the next row.

EXPERT: Shapes are drawn all over the screen at random until there is no space left for another shape. The shapes are never touching each other.

Tutorial: Speed

When drawing a larger pattern you might want to increase the turtle's speed.

1. The slowest speed is a speed of 1

t.speed(1)

2. Increasing the number increases the speed.

3. There is one exception: A speed of 0 is the fastest of all

t.speed(0)

PROJECT: Each square is drawn at a different speed.

ADVANCED: The speed of drawing the squares in each row gradually gets faster from left to right.

EXPERT: Lighter colors are drawn faster than darker ones, but the time it takes for a shape to be drawn is always the same, no matter the type of shape.
